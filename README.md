# Materials for Mkdocs の試し

Bootstrap

```bash
$ python -m venv .venv
$ source .venv/bin/activate
$ pip install mkdocs-material
$ pip install mkdocs-static-i18n
```

Test, Build and Deploy

```bash
$ mkdocs serve
$ mkdocs build
$ mkdocs gh-deploy --force
```
